Setup Instructions

Step 1: Add Your OpenAI API Key

To use the AI Assistant, you need an OpenAI API key. This enables the GPT-4o-mini model for intelligent coaching suggestions.

  1. 1. Go to OpenAI API Keys
  2. 2. Create a new API key
  3. 3. Open js/utils/constants.js
  4. 4. Replace 'your-openai-api-key-here' with your actual API key

How to Use the AI Assistant

1

Access the Admin Dashboard

Click the "Admin" button and sign in with your Google account or admin credentials.

2

Find the AI Assistant Button

Look for the purple "AI Assistant" button next to each practice's "Edit" button.

3

Describe Your Team's Challenges

Be specific! Examples: "Players aren't watching the ball into their glove" or "Kids are having trouble running through first base."

4

Choose Your AI Action

Select from drill suggestions, description generation, or parent summaries based on your needs.

5

Use the Results

Copy suggestions to your clipboard, find videos for drills, or send parent summaries directly to your team communication.

Scenario 4: Base Running Issues
